Goals Unit Test

Test by pressing Tap to play, selecting a gender, inputting a current weight, choosing a goal,and inputting a number of pounds you would like to gain or lose (based on which goal you chose). To see which values the app has stored, there is a line of fine print text at the bottom labeled "Choices:" that should display your choices in the order they are on the page. The final number is the calculated goal weight which will be calculated when you press ENTER.

Example:
Gender: Female
Weight: 125
Goal: Gain Muscle
lbs to lose/gain: 5
Expected values:
Choices: Female,125,Gain Muscle,5,130
